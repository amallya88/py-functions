The Assignment¶

Take a ZIP file) of images and process them, using a library built into python that you need to learn how to use. A ZIP file takes several different files and compresses them, thus saving space, into one single file. The files in the ZIP file we provide are newspaper images (like you saw in week 3). Your task is to write python code which allows one to search through the images looking for the occurrences of keywords and faces. E.g. if you search for "pizza" it will return a contact sheet of all of the faces which were located on the newspaper page which mentions "pizza". This will test your ability to learn a new (library), your ability to use OpenCV to detect faces, your ability to use tesseract to do optical character recognition, and your ability to use PIL to composite images together into contact sheets.

Each page of the newspapers is saved as a single PNG image in a file called images.zip. These newspapers are in english, and contain a variety of stories, advertisements and images. Note: This file is fairly large (~200 MB) and may take some time to work with, I would encourage you to use small_img.zip for testing.

class ImageData():
    """
    Class that holds a PIL image, a cv image (array) and a dictionary that contains
    locations of bounding boxes for faces and all the ocr text found in the image as string
    
    img_data: contains keys "txt", "faces" holds ocr text and 
    locations of faces (bounding boxes defined by [x0, y0, x1, y1] 2 corners of rect)
    """
            
    def __init__(self, orig_img, ocr_img, cv_img):
        self.img = orig_img
        self.ocr_img = ocr_img
        self.cv_img = cv_img
        self.img_data = {"txt": [], "faces": []}

def process_image_for_text(imagedata_obj):
    """ Given an ImageData object run the image through ocr and face detection
    and update the object's img_data dict
    """
    ocr_txt = pytesseract.image_to_string(imagedata_obj.ocr_img)
    ocr_txt = ocr_txt.replace("\n", ' ')
    imagedata_obj.img_data["txt"] = ocr_txt

def process_image_for_faces(imagedata_obj):    
    faces = face_cascade.detectMultiScale(imagedata_obj.cv_img,1.3,5)   
    for face in faces:
        imagedata_obj.img_data["faces"].append(np.add(face, [0, 0, face[0], face[1]]).tolist())

def draw_all_faces(imagedata_obj):
    """ Given an ImageData object return a contact sheet (Image) of all the faces found in the image 
    cropepd to 120x120 thumbnails. 
    """
    faces = imagedata_obj.img_data["faces"]  # list of lists
    size_thumb = 120  # max size of face thumbnail
    mode = imagedata_obj.img.mode
    
    if(len(faces) == 0):
        return None
    
    # iterate through bound boxes of faces and get cropped faces
    crop_faces = []
    for x0, y0, x1, y1 in faces:        
        crop_faces.append(imagedata_obj.img.crop((x0, y0, x1, y1)))
       
    width = 5 * size_thumb
    height = ((len(faces) // 5) + (1 if len(faces) % 5 > 0 else 0)) * size_thumb
    contact_sheet = Image.new(mode, (width, height))
    
    x=0
    y=0    
    for face in crop_faces:
        # resize to thumb
        if face.width > size_thumb or face.height > size_thumb:
            face.thumbnail((size_thumb, size_thumb), resample=Image.LANCZOS)
        
        # Lets paste the current image into the contact sheet
        contact_sheet.paste(face, (x, y) )
        # Now we update our X position. If it is going to be the width of the image, then we set it to 0
        # and update Y as well to point to the next "line" of the contact sheet.
        if x + size_thumb == contact_sheet.width:
            x = 0
            y += size_thumb
        else:
            x += size_thumb
    return contact_sheet
